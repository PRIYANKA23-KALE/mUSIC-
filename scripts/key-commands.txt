
<p><b> Navigation commands <b>

esc	unfocus graphic element(s)	.

<b> More commands (+<i>alt</i> key) <b>
g	display SVG text for current page	/commands/alt-g
o	toggle original clefs	/commands/alt-o

<p><b> Note editing commands </b></p>

p	toggle pedal start before	.
P	toggle pedal stop after	.
q	toggle grace-note style	.
s	add slur to next note	/graphic/slurs
@	toggle mark	/graphic/marks
&uarr;/&darr;	next/previous harmonic note	/graphic/pitch
&larr;/&rarr;	next/previous melodic note	/graphic/pitch
shft-&uarr;/&darr;	up/down step	/graphic/pitch
# shft-&uarr;/&darr;	up/down interval	/graphic/pitch
ctrl-&uarr;/&darr;	up/down octave	/graphic/pitch

<p><b> Rest editing commands </b></p>

y	make rest invisible	.

<p><b> Slur editing commands </b></p>

&larr;	move slur start 1 note left	.
&rarr;	slur start 1 note right	.
#&larr;	slur start # notes left	.
#&rarr;	slur start # notes right	.
<nobr>shft-&larr;</nobr>	slur end left	.
<nobr>shft-&rarr;</nobr>	slur end right	.
<nobr># shft-&larr;</nobr>	slur end # notes left	.
<nobr># shft-&rarr;</nobr>	slur end # notes right	.
